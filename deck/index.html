The problem

AI agents now manage $4.2B+
and zero of it is insurable.

Market failures

Flash crashes, oracle drift, liquidity drains. Agent acts correctly — environment shifts.

Agent failures

Prompt injection, model drift, faulty decision logic. The agent itself misbehaves.

Infra failures

RPC outages, key compromise, oracle downtime. External systems break.

"Traditional insurers don't price agent risk. DeFi insurers exclude AI losses. Manual claims take 30-90 days. By payout, the protocol is dead."

The solution

Parametric insurance, written for agents.

On-chain triggers

Loss thresholds, oracle deviation, anomaly detection. No claim adjustment. No human in the loop.

USDC payout in 60s

Smart contract executes the payout the moment the trigger condition resolves on-chain.

AI-native pricing

Premiums scored from agent behavior, not actuarial templates. Better data → better pricing → market share.

Multi-chain by default

Solana, Base, Arbitrum, BSC. Coverage follows the capital, not the chain.
