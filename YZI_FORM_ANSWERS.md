AgentInsurance solves this with parametric coverage:
- On-chain triggers (loss > threshold, oracle deviation, anomaly detection)
- Smart contract execution (USDC payout in <60 seconds)
- AI-native risk scoring (premiums based on agent behavior history)
- Multi-chain (Solana, Base, Arbitrum, BSC)
- Composable primitive (other protocols build on top)

Result: Agent operators can deploy 10-100x more capital because tail risk is hedged. DeFi protocols can integrate AI agents without bearing autonomous-failure liability. The agentic economy gets the financial plumbing it needs to scale.

3. Composability: open source contracts get integrated by other protocols, creating network effects.
4. Recruitment: open source attracts the engineers we want to hire.
5. Marketing: developers discover us through GitHub, fork our code, then integrate.
6. Moat alignment: our moat is data + brand + integrations, NOT proprietary code. Open source amplifies our actual moats.

WHY NOT TOTAL OPEN SOURCE:
Risk model weights stay closed because:
- Adversarial actors could game triggers if scoring is fully transparent
- Years of policy data become competitive defensibility
- Industry standard (Lloyd's, Munich Re) keeps actuarial models proprietary

Our open-source strategy mirrors successful Web3 protocols (Uniswap, Aave): open contracts, proprietary curation. It works because Web3 users trust verifiable code over closed promises, and developers integrate what they can read.

Bottom line: we ship in public, build in public, open in public. The next decade of Web3 insurance should be built openly and we want to set that standard.
