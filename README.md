# AgentInsurance

**Parametric Insurance for AI Agent Failures**

The first decentralized insurance protocol protecting businesses from autonomous AI agent errors, powered by smart contracts and real-time verification.

---

## 🎯 The Problem

- AI agents make mistakes—wrong trades, bad decisions, data breaches
- Traditional insurance doesn't cover AI agents
- Manual claims take weeks to process
- **$4.2B lost to AI agent failures in 2025**

## 💡 The Solution

AgentInsurance provides:
- ✅ **Agent-Native Coverage** - Insurance designed for AI agents
- ⚡ **Instant Payouts** - Smart contracts verify and pay automatically
- 🔐 **Parametric Triggers** - Pre-defined conditions trigger claims
- 📊 **Risk Scoring** - On-chain reputation system
- 🌐 **Cross-Chain** - Works across all major chains
- 💎 **Stablecoin Native** - USDC/USDT premiums and payouts

## 🏗️ Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **UI Components:** Custom (inspired by shadcn/ui + Aceternity UI)
- **Smart Contracts:** Solidity (Ethereum/Solana)
- **Language:** TypeScript

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.
